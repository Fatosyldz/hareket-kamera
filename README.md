# Web kamerası ile gerçek zamanlı hareket algılayan eğlenceli bir React + Vite uygulaması

## Özellikler
- Kameradan gelen görüntüde hareket algılar
- Tarayıcıda direkt çalışır (kamera izni gerekir)
- Hafif ve çok hızlı (Vite sayesinde)
- Basit ve anlaşılır kod

## Nasıl Çalıştırılır?

git clone https://github.com/fatosyldz/hareket-kamera.git
cd hareket-kamera
npm install
npm run dev

Tarayıcıda http://localhost:5173 aç → kamera izni ver → hareket yap ve test et!

## Kullanılan Teknolojiler
- React
- Vite
- JavaScript (getUserMedia API + Canvas)
